# VS Code Profiles

A curated collection of Visual Studio Code profiles.

## Profiles

### Minimal UI

Located in `profiles/minimal_ui/`.

Includes:
- Vira Palenight High Contrast
- Cascadia Code PL (primary font)
- Custom keybindings

---

Profiles include only extensions that define their identity. What's included in `Minimal UI` profile? UI and theme extensions only

### Explicitly excluded
These are intentionally **not included** and should live in separate profiles.

#### Python / Data Science
- `ms-python.python`
- `ms-python.debugpy`
- `ms-python.vscode-pylance`
- `ms-toolsai.jupyter*`

#### C / C++
- `ms-vscode.cpptools*`
- `ms-vscode.cmake-tools`

#### Remote / SSH
- `ms-vscode-remote.remote-ssh`
- `ms-vscode-remote.remote-ssh-edit`
- `ms-vscode.remote-explorer`

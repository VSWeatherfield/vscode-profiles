# Font Installation (Windows)

## Required
### Cascadia Code (recommended)
Cascadia Code is an open-source font by Microsoft.

Install via:
- Microsoft Store (recommended), or
- https://github.com/microsoft/cascadia-code

After installation, restart VS Code.

## Optional
### Operator Mono
Operator Mono is a licensed font and is not included in this repository. If you own one (.otf file), feel free to use it

## Verification
In VS Code:
- Open Settings
- Ensure `editor.fontFamily` includes: Cascadia Code PL, Operator Mono, Consolas, monospace. 

If Operator Mono is not installed, VS Code will fall back gracefully.
